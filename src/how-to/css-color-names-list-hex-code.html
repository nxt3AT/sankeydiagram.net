About named-colors in CSS and their history

Named-colors are a collection of predefined colors specified in the CSS color specification (see the official CSS Color Module Level 4 specification document; https://drafts.csswg.org/css-color/#named-colors). These colors allow developers and users to easily use common hues without needing to know precise color codes (e.g. hex-code, rgb-code, or hsl-code).

Among the list of named-colors, 16 of them originated from the original VGA color-palette, a set of colors that was defined and used in early computer graphics. These 16 colors were later adapted into the HTML/CSS standard to bridge the transition from early graphical systems to more modern web development. The 16 VGA colors were combined with a big set of additional colors, totaling 147 color names that are now widely-supported in all browsers.

The later SVG (Scalable Vector Graphics) Standard, widely used for creating scalable images on the web and in other computing contexts, has also adapted the named-colors list. This allowing developrs and designers to also use the same color names as in web development for authoring SVG graphics instead of having to rely on specific color codes, and allowing to adapt already existing web designs into SVGs.

While not part of the official standard, there is also an additional named-color that is supported by major browsers including Chrome; "rebeccapurple" (#663399). This distinctive shade of purple was added by Eric A. Mayer, one of the driving forces of authoring and development of the initial CSS 1 standard, as a tribute to the memory of his second daughter.

All 147 standardized color names, plus the 1 additional "rebeccapurple", are supported by SankeyDiagram.net. Refer to the SankeyDiagram.net User Guide for more information on using these custom colors in your Sankey Charts.
